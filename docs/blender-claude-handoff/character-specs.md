# キャラクターグラフィック仕様

Blenderへ渡すためのキャラクター設定画は `character-graphics/` に保存しています。

## 画像一覧

| ファイル | 用途 |
| --- | --- |
| `character-graphics/player-businessman-turnaround.png` | 主人公サラリーマンの前・横・後ろ・走りポーズ |
| `character-graphics/enemy-commuters-reference.png` | 敵キャラ4種の参考設定 |
| `character-graphics/crowd-reaction-expressions.png` | 群衆リアクション、表情、駅員NPC参考 |
| `character-graphics/manifest.json` | 画像の役割とモデリングメモ |

## 共通スタイル

- チビ寄りのアニメ調3Dモデルを想定。
- 頭は大きめ、手足は短め、表情は大きく読む。
- TPS視点でも認識できるよう、髪型、バッグ、ネクタイ、帽子、スーツ色などでシルエットを分ける。
- 見た目はコミカル寄り。ゲームオーバー演出も直接的すぎる描写にしない。
- 足元原点は床中央。アニメーション時に靴が床から浮きすぎないようにする。

## 主人公

| 項目 | 内容 |
| --- | --- |
| モデル名 | `player-businessman.glb` |
| 役割 | プレイヤー操作キャラ |
| 服装 | 紺スーツ、白シャツ、青ストライプネクタイ、茶色革靴 |
| 顔 | 焦り顔、汗、限界に近い表情 |
| シルエット | 乱れたネクタイ、前傾姿勢、跳ねた黒髪 |

必要アニメーション:

- `idle`: 我慢して小刻みに震える
- `run`: 前傾姿勢で走る
- `turn_left`: 90度左旋回
- `turn_right`: 90度右旋回
- `hit`: ぶつかってよろける
- `limit`: 限界寸前で震える
- `clear`: トイレへ駆け込む
- `gameover`: 直接描写を避けた敗北ポーズ

## 群衆・駅員

| モデル名 | 役割 |
| --- | --- |
| `npc-crowd-variants.glb` | 背景群衆、リアクション |
| `npc-station-staff.glb` | ゲームオーバー時や駅構内の反応要員 |

リアクション:

- 嘲笑
- 哀れみ
- 驚き
- 気まずく目を逸らす
- ひそひそ話
- 駅員の困惑

## Blender制作メモ

- まずは主人公、ビジネスマン、旅行者の3モデルを優先する。
- 旅行者はキャリーケースを別メッシュにする。
- 敵キャラは共通ボーンを使い回せる体型にする。
- 表情はテクスチャ差分でもよいが、眉・目・口は別パーツ化するとリアクションが作りやすい。
- 低ポリで十分。質感はスーツ、革靴、髪、バッグが識別できればよい。
